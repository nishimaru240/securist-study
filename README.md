# SecuriST 試験対策アプリ

SecuriST 認定セキュアWebアプリケーション設計士の試験勉強用アプリ。
スライドビューアーと予想問題集の2つで構成され、スマートフォンで使えるシンプルな構成。
ジムのトレッドミルや通勤中などの隙間時間にも利用できます。

## デモ

| アプリ | URL | 内容 |
|--------|-----|------|
| スライドビューアー | https://nishimaru240.github.io/securist-study/ | 全8章・約154枚のスライドで知識を体系的に学習 |
| 予想問題集 | https://nishimaru240.github.io/securist-study/quiz.html | 全152問。章別演習と模試モード（30問）から選択 |

## 機能

### スライドビューアー（index.html）

- 手動スライド操作（進む・戻る・スワイプ）
- 自動進行モード（間隔を自由に設定）
- 目次からの章ジャンプ
- 文字サイズ調整（3段階）
- 最後に読んだ位置を自動保存

### 予想問題集（quiz.html）

- **章別モード**：選んだ章の全問題を出題（順序ランダム）
- **模試モード**：8章から3〜4問ずつバランスよく合計30問を出題
- 選択肢の順序も毎回シャッフル
- 解答後すぐに正誤・解説を表示
- 問題一覧から任意の問題へジャンプ可能
