# SecuriST 試験対策スライドアプリ

SecuriST 認定セキュアWebアプリケーション設計士の試験勉強用スライドビューアー。

スマートフォンで使えるシンプルな構成で、ジムのトレッドミルなどの利用中でも学習できます。

## デモ

https://nishimaru240.github.io/securist-study/

## 機能

- 手動スライド操作（進む・戻る・スワイプ）
- 自動進行モード（間隔を自由に設定）
- 目次からの章ジャンプ
- 文字サイズ調整（3段階）
- 最後に読んだ位置を自動保存
